Open Source

Help Build ApiScan

ApiScan is an open-source project built by and for the beekeeping community. Every contribution — whether code, documentation, translation, or feedback — makes a difference.

Ways to Contribute

You don't need to be a programmer to help. There's a meaningful role for everyone who cares about bees.

Report Bugs

Found something that doesn't work? Open a GitHub issue with device info, OS version, and steps to reproduce.

Open an issue on GitHub →

Submit Code

Fork the repo, create a branch, open a PR. Backend (Python), iOS (Swift), Android (Kotlin) — all welcome.

View the repository →

Improve Documentation

Help improve the in-app help text, the API contract, architecture notes, or this website.

Browse the docs folder →

Translate the App

Bilingual? Help add or improve a translation. iOS uses .strings files, Android uses XML resources — no programming required.

Join the translation discussion →

Test New Features

Join the beta program and give feedback before features ship. Your real-world perspective beats any automated test.

Join the beta discussion →

Spread the Word

Tell your beekeeping club or online community about ApiScan. Every new beekeeper strengthens the global dataset.

Share the download link →
